  4. Custom Cursor

    5. Any image bellow the resolution of 128x128 can be used as a cursor in html. Since there are different cursors for different purposes, like ⌛ looking cursor for loading pages, auto is specified after defining a single cursor. It wouldn't be required if a whole cursor pack is defined instead of just one.

    * {
      cursor: url(../images/cursor.png), auto;
    }

  5. Reactive Layout

    6. The css file, responsive.css, exists in the style directory in which different css rules for different screen dimenstions are specified.

    First section contains styling for tablet view, which is when: 675px < width < 1025px. These specified settings re-adjust the page for tablet view, and has been tested on iPads. Similarly, second section contains styling for phone view: width < 676px, and have been tested on iPhone 11pro.
